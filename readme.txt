A super lightweight and easy-to-use plugin for contacting with users on whatsapp!
If you want to easily contact with your users on your whatsapp, Chatzi is exactly made for you.

== Description ==

This super lightweight plugin adds a floating button on your website that your users can click on it and redirect easily to whatsapp web or whatsapp application on their mobile phones.

== Installation ==

1. Upload the plugin folder to your /wp-content/plugins/ folder, or install the plugin through the WordPress **Add plugin** page directly.
1. Go to the **Plugins** page and activate the plugin.
1. Access plugin settings under **Settings** menu from wordpress dashboard.

== Frequently Asked Questions ==

= Does this plugin slow down my website loading speed? =

No! Our first priority of developing chatzi plugin is to not affect the loading speed after activating plugin and keep chatzi a super lightweight plugin.

= Can I change the position of whatsapp button? =

Yes! You can simply change it through the plugin Settings.

= Where is plugin settings? =

After activating the plugin, go through **Settings** menu from your wordpress dashboard and you can see **Chatzi Settings**
